About Us

Coding4Kidz is a non-profit organization created at Del Norte High School with the mission to teach kids basic coding and introduce them to the STEM field. Our sessions typically last about 2 months and cover Scratch, Python, and Java programming.

Mission Statement

"We created Coding4Kidz to introduce coding to younger students so that they can be better prepared for the growing technological field."

Our Programs

PyGame

Learn game development with Python and PyGame! Create your own 2D games while learning programming fundamentals.

  • Basic Python programming
  • Game mechanics and physics
  • Graphics and animation
  • Sound and music integration

Scratch

Perfect for beginners! Learn programming concepts through visual block-based coding.

  • Visual programming
  • Interactive stories
  • Simple games
  • Creative animations

Roblox Lua

Build and program your own Roblox games! Learn Lua scripting to create interactive 3D experiences.

  • Lua programming basics
  • 3D game design
  • Multiplayer game mechanics
  • Roblox Studio tools
