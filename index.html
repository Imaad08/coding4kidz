About Us

Coding4Kidz is a non-profit organization created at Del Norte High School with the mission to teach kids basic coding and introduce them to the STEM field. Our sessions typically last about 2 months and cover Scratch, Python, and AI Application.

Mission Statement

"We created Coding4Kidz to introduce coding to younger students so that they can be better prepared for the growing technological field."

Our Programs

Scratch

Perfect for beginners! Learn programming concepts through visual block-based coding.

  • Visual programming
  • Interactive stories
  • Simple games
  • Creative animations
  • Prerequisites: None

Python

Master Python from beginner to intermediate level! Build a strong foundation to prepare for AI Application development.

  • Python fundamentals and syntax
  • Data structures and algorithms
  • Functions and object-oriented programming
  • File handling and APIs
  • Prerequisites: None

AI Application

Build intelligent applications using AI APIs like Gemini! Learn to integrate artificial intelligence into your projects.

  • AI API integration
  • Gemini API basics
  • Building chatbots
  • Smart application development
  • Prerequisites: Intermediate Python
